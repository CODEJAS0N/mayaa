JSTL Format with Maya

formatNumber

記述

<span m:inject="fmt:formatNumber">1234567.890</span><br />
<span m:inject="fmt:formatNumber" fmt:value="${ 1234567.890 }">ダミーデータ</span><br />
整形: <span m:inject="fmt:formatNumber" fmt:pattern="正#,##0.00;負#,##0.00">123.789</span><br />
整形: <span m:inject="fmt:formatNumber" fmt:pattern="正#,##0.00;負#,##0.00">-123.789</span><br />
区切り: <span m:inject="fmt:formatNumber" fmt:groupingUsed="false">1234567.890</span><br />
通貨: <span m:inject="fmt:formatNumber" fmt:type="CURRENCY" fmt:currencySymbol="$">1234567.890</span><br />
少数桁: <span m:inject="fmt:formatNumber" fmt:maxFractionDigits="1">1234567.890</span><br />
整数桁: <span m:inject="fmt:formatNumber" fmt:maxIntegerDigits="3">1234567.890</span><br />
整数桁: <span m:inject="fmt:formatNumber" fmt:minIntegerDigits="3">12.3</span><br />
変数: <span m:inject="fmt:formatNumber" fmt:value="${ 1234567.890 }" fmt:var="number" fmt:scope="AT_END">ダミーデータ</span>
<span m:inject="c:out" fmt:value="$#36;{ number }" /><br />

説明

数値をフォーマットするためのカスタムタグ。
